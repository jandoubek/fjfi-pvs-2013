Intents - použití pro různé typy komunikací
- přímá aktivace AKTIVIT a SLUŽEB - teď udělej tohle // spusť třídu (něco jako volání)
- nepřímá aktivace AKTIVIT a SLUŽEB - co se zalová se určuje až při ladění
- šíření zpráv pro více odběratelů - (broadcats recievers),
  např. dochází baterie, přijde SMS - řekne se to všem - nebo téměř všem, i bezpečnostní prvky
- meziproces. komunikaci
- vytvoření poskytovatelů pro sdílených prostředků (DB, dat, internetu)
Pokaždé je to úplně jiný intent

Service - na pozadí
Aktivita - vizuální prvek

Stavy aktivit
- životní cyklus aplikací/procesů
aplikace Java SE v podstatě spuštění/ukončení main nebo podprogramu,
zde velmi zesložitěné, vztah i s dalšími aktivitami (např. nemá dostatek prostředků),
jedna apliakce umí killnout jinou

samotný konstruktor aktivity by se dělat neměl, jen tyto události
 onCreate - vzít prostředky,
 onStart - už je nahraná v paměti, předpokládám
 onResume - na popření
 onFreeze - na skoro pozadí (např. překrytí jinou)
 onStop - na pozadí, nelze s ní interagovat
 onDestroy - kill aplikace, uvolnit prostředky



Download, instalace, konfigurace
================================
1) Java SE Development Kit
http://www.oracle.com/technetwork/java/javase/downloads/jdk-7u2-download-1377129.html
 - Stáhnout, nainstalovat

2) Vývojové prostředí Ecplipse
http://www.eclipse.org/downloads/
Ecplipse classic nebo pro Javu (pokud možno všetně Java SDK)
 - Stáhnout, rozzipovat, překopírovat, spustit

 Vhodné nakonfirovat:
   1) vybrat umístění vlastních projektů (workspace)
   2) Run - Run Configurations

3) ADT Plugin for Eclipse - tzn. emulátor Android pro Eclipse

 - Přímo v menu prostředí Eclipse :
    * Help - Install new software
    * Developer Tools
    * zadání adresy:
      https://dl-ssl.google.com/android/eclipse/
    * výběr správné verze Android (nejlépe API 8 = Android 2.2)
    * stažení a instalace
